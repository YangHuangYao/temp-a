# vue-template-admin

先安装脚手架 ymc-vue-cli
全局 npm i ymc-vue-cli –g

## create project

ymc init template projectName(要创建的项目名称)

## Project setup

```
yarn install
```

### Compiles and hot-reloads for development

```
yarn run serve
```

### Compiles and minifies for production

```
yarn run build
```

### Run your tests

```
yarn run test
```

### Lints and fixes files

```
yarn run lint
```
